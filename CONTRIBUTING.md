# Contributing to OpenAnimal

Thanks for your interest in contributing. This project is open source and
welcomes pull requests, bug reports, and ideas.

## Quick start

1. Fork the repository and create a feature branch.
2. Install dependencies:

   ```bash
   python -m pip install -r requirements.txt
   ```

3. Run tests:

   ```bash
   python -m pytest
   ```

4. Make your changes and keep commits focused.
5. Open a pull request with a clear description of the change.

## Code style

- Prefer small, readable functions.
- Keep changes scoped to one intent per commit.
- Update docs when behavior or configuration changes.

## Reporting issues

If you find a bug or have a feature request, please open a GitHub issue with:

- Steps to reproduce (if applicable)
- Expected behavior and actual behavior
- Relevant logs or screenshots

## Community

Please read and follow the Code of Conduct in CODE_OF_CONDUCT.md.
